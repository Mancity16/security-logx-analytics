**Built with:** Python | June 2026

## Project Overview
This repository contains a Python application I developed to bridge my professional background in physical security with my new Computer Science skills.

The script processes, filters, and statistically evaluates simulated physical access control data from high-security facilities. It analyzes timestamps, employee IDs, locations, and access status codes `200` = Granted, `403` = Denied to identify patterns and potential system vulnerabilities.

## Purpose & Academic Relevance
I created this project as part of my preparation for a B.Sc. in Computer Science at IU International University of Applied Sciences, Germany. 

After 7 years working in the physical security sector since 2018, I wanted to transition into the technical side of security. This project applies concepts from my June 2026 coursework in Python, Statistics, and IT Security to a domain I already know.

## Key Features
- Parse and clean CSV access log data using Python
- Filter records by status code to isolate denied access attempts 
- Statistical evaluation of access patterns by time and location
- Output summary reports to support infrastructure review

## How to Run
1.  Clone the repository: `git clone https://github.com/Mancity16/security-logx-analytics.git`
2.  Run the script: `python analyzer.py`
